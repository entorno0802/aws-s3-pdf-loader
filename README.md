# aws-s3-pdf-loader
AWS-S3-PDF-LOADER
# how to run
1. Copy .env.example and rename it as .env.
   
   ```cp .env.example .env```
2. Open .env file and set your own AWS_ACCESS_KEY_ID and AWS_SECRET_ACCESS_KEY.
   
3. Run the app.
   
   ```python app.py```
